## 📊 Guard Pattern (Used Everywhere)

```csharp
// Check this
if (HttpContext == null) {
    Log error → Return null
}

// Then check that
if (HttpContext.Session == null) {
    Log error → Return null
}

// Then check the value
if (accountId == null || accountId == 0) {
    Log warning → Deny access
}

// If we reach here, it's safe
return value  // All guards passed!
```

## 🚨 Common Issues & Solutions

### Issue: "Connection Refused"
**Check Logs For:**
```
❌ Connection string 'PethubContext' not found
```
**Fix:** Connection string in `appsettings.json`

### Issue: "Session is null"
**Check Logs For:**
```
❌ Session middleware not configured or disabled
```
**Fix:** `app.UseSession()` in `Program.cs`

### Issue: "Keep redirecting to login"
**Check Logs For:**
```
❌ No valid session found, redirecting to Login
```
**Fix:** Session expired or not set during login
